# Feature Ideas

Unscoped ideas for future consideration. None of these are committed.

- **Chemistry preset profiles**: config.toml could include named presets such as `twist_duplex`, `umi_tools_standard`, etc. Users select a preset and override individual fields as needed.
- **Batch mode**: Run multiple samples with one config file. Useful for cohort-level benchmarking and clinical workflows.
- **Validation mode**: Compare output against a truth VCF automatically. Reports sensitivity, precision, and F1 without a separate script.
