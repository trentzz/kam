# docs

Documentation for the kam project.

## Contents

| Folder | Purpose |
|---|---|
| `research/` | Prior art and background reading: tool landscape, chemistry details, algorithm analysis |
| `manual/` | End-user documentation: installation, usage, configuration, CLI reference |
| `examples/` | Example configs and worked usage examples (moved from repo root `examples/`) |
| `benchmarking/` | Benchmark definitions, configs, scripts, and small result files for each variant class |
| `paper/` | LaTeX source, figures, and supplementary materials for the publication |
| `project/` | Internal project material: task tracking, devmanual, investigations, experiments, vision |

Large generated files (simulation FASTQs, model weights, PDFs) live in `bigdata/` at the repo root, not here. See `project/devmanual/nextcloud.md` for details.
