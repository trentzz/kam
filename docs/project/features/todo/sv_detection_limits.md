# SV Detection Limits at Ultra-Low VAF

**Status**: todo
**Epic**: SV-ULTRAVAF

## Summary

Establish the detection floor for each SV type at ultra-low VAF levels
(0.01-0.04%) using synthetic data at 5000x coverage.

## VAF Levels

0.01%, 0.02%, 0.03%, 0.04% (extending the existing 0.05-10% range)

## Deliverables

- Detection limit curves per SV type (discovery and tumour-informed modes)
- Minimum detectable VAF per type at 80% sensitivity
- Recommendations for clinical detection thresholds
