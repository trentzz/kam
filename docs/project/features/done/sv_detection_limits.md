# SV Detection Limits at Ultra-Low VAF

**Status**: done
**Epic**: SV-ULTRAVAF

## Summary

Established the detection floor for each SV type at ultra-low VAF levels
(0.01-0.04%) using synthetic data at 5000x coverage.

## VAF Levels

0.01%, 0.02%, 0.03%, 0.04% (extending the existing 0.05-10% range)

## Deliverables

- [x] Detection limit curves per SV type (discovery and tumour-informed modes)
- [x] Minimum detectable VAF per type at 80% sensitivity
- [x] Recommendations for clinical detection thresholds
- [x] User-facing guide: `docs/manual/guides/sv-detection-limits.md`
