# project

Internal project material for kam. This directory is for contributors and maintainers, not end users.

## Contents

| Folder | Purpose |
|---|---|
| `devmanual/` | Developer reference: architecture, design decisions, benchmarking how-to, task workflow, Nextcloud |
| `tracking/` | Kanban task board: todo, in-progress, and done tasks |
| `experiments/` | Experiment records: hypothesis, design, and outcome for each research experiment |
| `investigations/` | Diagnostic investigation writeups: symptom, root cause, and resolution |
| `features/` | Feature specs: proposed, in-progress, and completed features |
| `vision/` | Project direction: goals, paper focus, result priorities |
| `review/` | Review cycle outputs: code quality, performance, correctness assessments |
