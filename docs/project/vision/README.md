# vision

Project direction documents for kam. These define goals, paper focus, result priorities, and feature direction. Run `/guide` to update them.

## Subfolders

| Folder | Contents |
|---|---|
| `project/` | Goals, audience, and scope |
| `paper/` | Paper focus, structure, and target figures |
| `results/` | Result priorities, comparisons, and planned experiments |
| `features/` | Feature priorities and deferred items |
| `decisions/` | Append-only log of design decisions |
